# API Reference

DeepCrate includes an interactive API reference powered by [Scalar](https://scalar.com/). It is served directly by the DeepCrate server so you can explore endpoints and make live requests against your running instance.

## Accessing the API Docs

Once DeepCrate is running, navigate to:

```
http://<your-host>:8080/api/v1/docs
```

## OpenAPI Spec

The raw OpenAPI JSON spec is available at:

```
http://<your-host>:8080/api/v1/openapi.json
```
